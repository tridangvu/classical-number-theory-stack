# Résultats

Ce dossier contient les fiches synthétiques des résultats mathématiques.

Une fiche de résultat doit présenter :

- l'énoncé moderne ;
- les formulations historiques ;
- la motivation ;
- les dépendances ;
- les conséquences ;
- les liens vers les dossiers de preuves.

Les démonstrations longues doivent être placées dans `proofs/`.
