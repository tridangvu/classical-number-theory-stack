# Bibliographie minimale

## Objectif

Ce fichier rassemble les références de départ du projet.

Les sources sont classées en quatre catégories :

1. sources primaires historiques ;
2. synthèses historiques ;
3. manuels modernes ;
4. références avancées.

---

# I. Sources primaires historiques

## Fermat

- Fermat, correspondance et observations arithmétiques.
- À utiliser surtout pour :
  - descente infinie ;
  - sommes de deux carrés ;
  - dernier théorème de Fermat ;
  - problèmes diophantiens.

## Euler

- Euler, travaux arithmétiques sur les résidus, les sommes de puissances, les produits infinis.
- À utiliser surtout pour :
  - petit théorème de Fermat ;
  - théorème des deux carrés ;
  - premiers produits eulériens ;
  - nombres de Bernoulli.

## Lagrange

- Lagrange, travaux sur les formes quadratiques et la théorie des nombres.
- À utiliser surtout pour :
  - formes quadratiques ;
  - réduction ;
  - équations diophantiennes ;
  - fractions continues.

## Legendre

- Legendre, Essai sur la théorie des nombres.
- À utiliser surtout pour :
  - symbole de Legendre ;
  - résidus quadratiques ;
  - réciprocité quadratique ;
  - formes quadratiques.

## Gauss

- Gauss, Disquisitiones Arithmeticae, 1801.
- À utiliser surtout pour :
  - congruences ;
  - réciprocité quadratique ;
  - formes quadratiques binaires ;
  - composition ;
  - genres ;
  - cyclotomie.

## Dirichlet

- Dirichlet, travaux sur les progressions arithmétiques.
- Dirichlet-Dedekind, Vorlesungen über Zahlentheorie.
- À utiliser surtout pour :
  - caractères ;
  - fonctions $L$ ;
  - théorème des progressions arithmétiques ;
  - unités ;
  - nombres de classes.

## Kummer

- Kummer, travaux sur les nombres idéaux et les corps cyclotomiques.
- À utiliser surtout pour :
  - dernier théorème de Fermat ;
  - corps cyclotomiques ;
  - nombres idéaux ;
  - nombres réguliers ;
  - nombres de Bernoulli.

## Dedekind

- Dedekind, suppléments aux Vorlesungen über Zahlentheorie de Dirichlet.
- À utiliser surtout pour :
  - entiers algébriques ;
  - idéaux ;
  - anneaux d'entiers ;
  - factorisation unique des idéaux ;
  - groupes de classes.

## Kronecker

- Kronecker, travaux sur la multiplication complexe et le Jugendtraum.
- À utiliser surtout pour :
  - construction explicite d'extensions abéliennes ;
  - multiplication complexe ;
  - analogie cyclotomie / fonctions elliptiques.

## Hilbert

- Hilbert, Zahlbericht, 1897.
- À utiliser surtout pour :
  - synthèse de la théorie algébrique des nombres ;
  - corps de nombres ;
  - idéaux ;
  - unités ;
  - ramification ;
  - lois de réciprocité ;
  - transition vers le corps de classes.

# II. Synthèses historiques modernes

## Edwards

- Harold M. Edwards, Fermat's Last Theorem.
- Très utile pour :
  - Fermat ;
  - Euler ;
  - Kummer ;
  - corps cyclotomiques ;
  - nombres idéaux ;
  - histoire technique du dernier théorème de Fermat.

## Weil

- André Weil, Number Theory: An Approach Through History from Hammurapi to Legendre.
- Très utile pour :
  - histoire longue de la théorie des nombres ;
  - Fermat ;
  - Euler ;
  - Lagrange ;
  - Legendre.

## Cox

- David A. Cox, Primes of the Form $x^2 + ny^2$.
- Très utile pour :
  - formes quadratiques ;
  - composition ;
  - genres ;
  - corps quadratiques ;
  - corps de classes sous une forme accessible.

# IV. Règle d'utilisation

Chaque source importante devra recevoir progressivement une fiche dans `sources/`.

Une fiche de source doit indiquer :

- référence complète ;
- statut : primaire, secondaire, moderne, avancée ;
- période couverte ;
- concepts couverts ;
- résultats couverts ;
- utilité précise pour le projet.
