# Taxonomie initiale du projet

## Objectif

Ce fichier sert à organiser les grands blocs intellectuels du projet.

Le but n'est pas de figer définitivement la structure, mais de disposer d'une première carte pour classer :

- les résultats ;
- les personnes ;
- les problèmes ;
- les concepts ;
- les sources ;
- les dépendances historiques et mathématiques.

---

# I. Axes principaux

## 1. Arithmétique fermatienne et eulérienne

### Objets

- nombres entiers ;
- divisibilité ;
- congruences élémentaires ;
- résidus ;
- puissances modulo un entier ;
- équations diophantiennes ;
- descente infinie.

### Problèmes moteurs

- Quels entiers sont sommes de deux carrés ?
- Quand une équation diophantienne a-t-elle une solution ?
- Comment contrôler les puissances modulo un nombre premier ?
- Pourquoi la descente infinie permet-elle de démontrer des impossibilités ?

### Personnes centrales

- Fermat ;
- Euler ;

### Résultats fondamentaux

- Petit théorème de Fermat.

- Identité des deux carrés.

Question directrice :

Exemples :

- descente infinie ;
- sommes de deux carrés ;
- dernier théorème de Fermat ;
- factorisation dans des anneaux plus grands ;
- corps cyclotomiques ;
- régularité de Kummer.

## Fil rouge B — Réciprocité

Question directrice :

Pourquoi les lois de réciprocité sont-elles le cœur caché de l'arithmétique classique ?

Exemples :

- réciprocité quadratique ;
- réciprocité cubique ;
- réciprocité biquadratique ;
- lois de réciprocité supérieures ;
- corps de classes comme forme moderne ultime.

## Fil rouge C — Factorisation

Question directrice :

Que devient la factorisation unique quand on quitte \(\mathbf Z\) ?

- factorisation dans les anneaux cyclotomiques ;
- échec de la factorisation unique ;
- nombres idéaux de Kummer ;
- idéaux de Dedekind ;
- groupes de classes.

# V. Règle pratique

Chaque fois qu'une nouvelle note est créée, elle doit idéalement indiquer :

- à quel axe elle appartient ;
- quels résultats elle utilise ;
- quels résultats elle prépare ;
- quelle personne ou quelle source historique elle concerne ;
- son statut : brouillon, esquisse, stable, à vérifier.
