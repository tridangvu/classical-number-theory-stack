# Formalisme

Ce dossier contient les fiches de formalisme mathématique.

Ces fiches expliquent les outils utilisés dans les résultats :

- congruences ;
- résidus quadratiques ;
- groupes ;
- anneaux ;
- anneaux euclidiens ;
- anneaux principaux ;
- anneaux factoriels ;
- entiers de Gauss ;
- entiers algébriques ;
- anneaux de Dedekind ;
- idéaux ;
- corps de nombres ;
- fonctions L ;
- formes quadratiques.

Ces fiches doivent être réutilisables dans plusieurs résultats.
