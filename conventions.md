# Conventions du projet

## Langue

Langue principale : français.

## Format des fichiers

Les fichiers sont écrits en Markdown.

## Nommage des fichiers

Utiliser des noms courts, explicites, en minuscules, avec des tirets.

Exemples :

- `fermat-little-theorem.md`
- `quadratic-reciprocity.md`
- `gauss-disquisitiones-arithmeticae.md`

## Types de fiches

- Résultat mathématique : utiliser `templates/result-entry.md`.
- Personne : utiliser `templates/person-entry.md`.
- Source : utiliser `templates/source-entry.md`.
- Sujet : utiliser `templates/topic-entry.md`.

## Niveau de rigueur

Chaque résultat important doit idéalement contenir :

- un énoncé moderne clair ;
- le contexte historique ;
- les motivations ;
- les dépendances conceptuelles ;
- une preuve ou une esquisse ;
- des exercices ;
- des références.
