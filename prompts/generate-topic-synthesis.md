# Prompt — fiche thématique de synthèse

Génère une fiche thématique de synthèse pour :

Thème : [NOM DU THÈME]
Chemin souhaité : `topics/[nom-du-theme].md`
Période : [période]

Objectif :
Créer une fiche-pivot qui explique la logique d’ensemble d’un bloc du projet, sans remplacer les fiches détaillées.

Structure attendue :
1. Métadonnées.
2. Vue d’ensemble.
3. Problèmes historiques.
4. Chaîne logique.
5. Résultats principaux.
6. Formalismes nécessaires.
7. Personnages et contributions.
8. Carte des dépendances.
9. Parcours de lecture conseillé.
10. Fiches à créer.

Contraintes :
- français ;
- Markdown ;
- LaTeX avec \( ... \) et \[ ... \) uniquement ;
- pas de dollars.
