# Batch — réciprocité quadratique

Tu travailles dans le dépôt `classical-number-theory-stack`.

Génère le bloc initial autour de la réciprocité quadratique.

Créer les fichiers suivants :

1. `results/quadratic-reciprocity.md`
2. `proofs/quadratic-reciprocity-proofs.md`
3. `concepts/legendre-symbol.md`
4. `concepts/jacobi-symbol.md`
5. `concepts/gauss-lemma.md`
6. `concepts/quadratic-forms-binary.md`
7. `people/legendre.md`
8. `people/gauss.md`
9. `topics/quadratic-reciprocity-cluster.md`

Contraintes importantes :
- `results/quadratic-reciprocity.md` doit rester une fiche courte : énoncé, contexte, motivation, idée, conséquences.
- Les preuves longues doivent aller dans `proofs/quadratic-reciprocity-proofs.md`.
- Le dossier de preuves doit comparer au moins :
  1. preuve par le lemme de Gauss ;
  2. preuve par sommes de Gauss ;
  3. preuve par formes quadratiques ;
  4. preuve via corps cyclotomiques ou caractères, seulement comme perspective moderne.
- Ne pas prétendre démontrer complètement toutes les preuves si le fichier devient trop long.
- Ajouter les résultats utilisés sans preuve complète.

Contraintes formelles :
- français ;
- Markdown ;
- LaTeX avec \( ... \) et \[ ... \) ;
- pas de dollars ;
- liens internes Markdown proposés ;
- section “À faire” dans chaque fichier.
