# Prompt — fiche de formalisme

Génère une fiche de formalisme pour le sujet suivant :

Formalisme : [NOM DU FORMALISME]
Chemin souhaité : `concepts/[nom-du-formalisme].md` ou `formalism/[nom-du-formalisme].md`
Période d’apparition historique : [période]
Utilisé par : [résultats principaux]
Dépendances : [liste]

Utilise l’esprit du template `templates/formalism-entry.md`.

La fiche doit contenir :
1. Métadonnées.
2. Rôle dans le projet.
3. Définitions.
4. Exemples fondamentaux.
5. Propositions de base.
6. Théorèmes structurants.
7. Preuves essentielles.
8. Pièges et confusions.
9. Liens internes.
10. Exercices.
11. Références.

Contraintes :
- français ;
- Markdown ;
- LaTeX avec \( ... \) et \[ ... \) uniquement ;
- pas de dollars.
