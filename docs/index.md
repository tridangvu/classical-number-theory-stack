# Classical Number Theory Stack

Projet encyclopédique personnel et évolutif consacré à l'histoire technique de la théorie des nombres, de Fermat à Hilbert.

## Objectif

Construire progressivement une base de connaissances structurée en Markdown contenant :

- des résultats mathématiques formulés proprement ;
- des motivations historiques ;
- des preuves ou esquisses de preuves ;
- des dépendances conceptuelles ;
- des exercices ;
- des références ;
- des timelines ;
- éventuellement plus tard un graphe de dépendances et un site web.

## Périmètre initial

De Fermat à Hilbert, avec un premier axe autour de :

- Fermat ;
- Euler ;
- Lagrange ;
- Legendre ;
- Gauss ;
- Jacobi ;
- Dirichlet ;
- Kummer ;
- Dedekind ;
- Kronecker ;
- Hilbert.
