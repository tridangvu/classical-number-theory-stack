# Dossiers de preuves

Ce dossier contient les fiches indépendantes consacrées aux démonstrations.

Règle centrale :

Si la preuve d'un résultat est significativement plus longue que son énoncé, alors le résultat doit avoir un dossier de preuves séparé.

Exemples :

- `two-squares-theorem-proofs.md`
- `quadratic-reciprocity-proofs.md`
- `dirichlet-unit-theorem-proofs.md`
- `ideal-factorization-proofs.md`

Un dossier de preuves peut contenir :

- preuve historique ;
- preuve élémentaire ;
- preuve par congruences ;
- preuve par formes quadratiques ;
- preuve algébrique ;
- preuve analytique ;
- preuve moderne ;
- comparaison des méthodes.
